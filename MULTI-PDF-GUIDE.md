# 複数PDF統合デジタル回覧板機能 使用ガイド

複数のPDFファイルから記事を抽出して、1つのデジタル回覧板（月号）としてまとめる機能の使い方を説明します。

## 機能概要

### 実現できること

- 複数のPDFファイルを1つのデジタル回覧板（例: 2025年1月号）にまとめる
- 各PDFから記事を抽出して累積的に追加
- 登録済みPDFの一覧表示
- 全記事をまとめて確認・保存

### 利用シーン

**例: 2025年1月号の作成**

```
2025年1月号
├─ 関ヶ谷だより.pdf → 3件の記事
├─ 会報ふれあい.pdf → 5件の記事
└─ 各種お知らせ.pdf → 2件の記事
合計: 10件の記事
```

## 使用方法

### Step 1: デジタル回覧板を作成

1. 管理画面を開く（http://localhost:3000/）
2. 「デジタル回覧板」をクリック
3. 「デジタル回覧板」タブをクリック
4. **タイトル**を入力（例: 2025年1月号）
5. 「デジタル回覧板を作成」ボタンをクリック

画面が切り替わり、編集モードになります。

### Step 2: 1つ目のPDFを追加

1. 「PDFを追加」セクションで「クリックしてPDFを選択」をクリック
2. 1つ目のPDF（例: 関ヶ谷だより.pdf）を選択
3. 「記事を抽出して追加」ボタンをクリック
4. 20-60秒待つ（AI処理中）
5. アラート「3件の記事を追加しました（処理時間: XX秒）」

**登録済みPDF一覧に追加されます:**
```
登録済みPDF (1件)
✓ 関ヶ谷だより.pdf - 3件の記事
```

**記事一覧に表示されます:**
```
抽出された記事 (3件)
- どんど焼き開催のお知らせ
- 会館の臨時休館について
- ...
```

### Step 3: 2つ目のPDFを追加

1. 再度「クリックしてPDFを選択」をクリック
2. 2つ目のPDF（例: 会報ふれあい.pdf）を選択
3. 「記事を抽出して追加」ボタンをクリック
4. 20-60秒待つ
5. アラート「5件の記事を追加しました / 合計: 8件の記事」

**登録済みPDF一覧が更新されます:**
```
登録済みPDF (2件)
✓ 関ヶ谷だより.pdf - 3件の記事
✓ 会報ふれあい.pdf - 5件の記事
```

**記事一覧が更新されます:**
```
抽出された記事 (8件)
- どんど焼き開催のお知らせ
- 会館の臨時休館について
- 新春俳句会の報告
- 資源ごみ収集日の変更
- 防災訓練のお知らせ
- ...
```

### Step 4: 3つ目以降も同様に追加

何度でもPDFを追加できます。各追加ごとに記事が累積されます。

### Step 5: 記事を確認・保存

1. 記事一覧で内容を確認
2. カテゴリフィルターで絞り込み
3. 詳細度を切り替え（見出し/簡潔/要約/全文）
4. 「記事を保存」ボタンをクリック

## 機能の特徴

### 1. 累積的な記事追加

- 新しいPDFを追加しても、既存の記事は消えません
- 記事は累積的に追加されます
- いつでも追加を中断して保存できます

### 2. 登録済みPDFの追跡

- どのPDFを登録したか記録
- 各PDFから何件の記事が抽出されたか表示
- アップロード時刻の記録

### 3. リセット機能

「リセット」ボタンで編集をやり直せます：
- Newsletter情報をクリア
- 累積記事をクリア
- 登録済みPDFリストをクリア

### 4. 柔軟な編集

Newsletter作成後でも：
- 何度でもPDFを追加可能
- 記事一覧をリアルタイムで確認
- カテゴリや優先度でフィルタリング

## トラブルシューティング

### PDFを追加できない

**原因:** デジタル回覧板が未作成

**対処:** 先にタイトルを入力して「デジタル回覧板を作成」をクリック

### 記事が上書きされる

**原因:** リセットボタンを押した、またはページをリロードした

**対処:** リセット前に「記事を保存」を実行

### 同じ記事が重複する

**原因:** 同じPDFを複数回アップロードした

**対処:** 登録済みPDF一覧を確認してから追加

## よくある質問

### Q1. 何個までPDFを追加できますか？

制限はありません。ただし、ブラウザのメモリ上限に注意してください（目安: 10個程度まで）。

### Q2. PDFの順序は関係ありますか？

記事の順序は優先度とカテゴリで自動的にソートされます。PDFの追加順序は関係ありません。

### Q3. 途中で中断できますか？

現在はブラウザのメモリのみで動作しているため、ページをリロードするとデータが消えます。「保存する」ボタンをクリックすれば、Supabaseに保存されます。

### Q4. 抽出された記事を編集できますか？

現在は編集機能は未実装です。Phase 2以降で実装予定です。
